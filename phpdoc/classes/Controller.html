      Method dispatcher for the controller.

      invoke(string $method, mixed[] $data) : mixed[]

      Invokes a method of the current module. Accepts two arguments: the method name and an array containing the argument list of the function to be called. The value returned consists of an array of two objects: a boolean flag representing success or failure and, the data returned by the invoked method.

      $args=array($arg1,...);
$resultset=$obj->invoke('method_name',$args);

      Class loader for a module.

      loadSubModule(string $method) : null | object

      Loads a sub-module class and returns an object of the sub-module class. The argument accepted is that of a method name which is registered with the current module.

      $mod=$obj->loadSubModule('method_name');
